# SistRUGBY-SLTC

Sistema de Gestión de Partidos y Estadísticas de Rugby  
**Santiago Lawn Tennis Club — Santiago del Estero, Argentina**

---

## Descripción

SistRUGBY-SLTC es una aplicación de escritorio desarrollada en Java SE 17 con 
persistencia de datos en MySQL 8.0, diseñada para centralizar la gestión operativa 
de la sección rugby del SLTC. Permite registrar jugadores, partidos y eventos 
deportivos, generando reportes estadísticos de apoyo a la toma de decisiones del 
cuerpo técnico.

## Tecnologías

- **Lenguaje:** Java SE 17
- **Base de datos:** MySQL 8.0
- **Conectividad:** JDBC (MySQL Connector/J)
- **Interfaz gráfica:** Java Swing
- **Metodología:** Proceso Unificado de Desarrollo (PUD)
